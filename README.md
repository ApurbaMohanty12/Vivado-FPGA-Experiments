
# FPGA Design using Vivado

## Lab Experiments Report

This repository contains FPGA lab experiments performed using Xilinx Vivado 2025.2.
It includes Verilog code, testbenches, simulation results, and schematic diagrams.

---

## Tools & Technologies

* Xilinx Vivado 2025.2
* Verilog HDL

---

## Experiments

### Experiment 1: Logic Gates & Combinational Circuits

* Half Adder
* Full Adder
* 2x1 MUX
* 4x1 MUX
* 2:4 Decoder

### Experiment 2: Data Flow Modelling

* Half Adder
* Full Adder
* 2x1, 4x1, 8x1 MUX
* 3:8 Decoder

### Experiment 3: Code Converters

* Binary to Gray
* Gray to Binary
* Binary to Excess-3
* Binary to Octal, Decimal, Hexadecimal, BCD

### Experiment 4: Structural Modelling

* Full Adder using Half Adder
* 8x1 MUX using smaller MUX
* 4-bit Adder

### Experiment 5: Behavioral Modelling

* 8x1 MUX
* 4:16 Decoder

### Experiment 6: Encoder Designs

* Encoder
* Priority Encoder

### Experiment 7: 4-bit ALU

* Arithmetic: ADD, SUB, MUL, DIV
* Logical: AND, OR, XOR, NOT
* Shift Operations

### Experiment 8: Flip-Flops

* D Flip-Flop
* SR Flip-Flop
* JK Flip-Flop
* T Flip-Flop

### Experiment 9: Asynchronous Flip-Flops

* D, JK, SR, T Flip-Flops

### Experiment 10: Counters

* Up Counter
* Down Counter
* Up-Down Counter

### Experiment 11: Shift Registers

* SISO
* SIPO
* PISO
* PIPO
